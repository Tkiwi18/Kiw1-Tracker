Tracker for OoT X MM Combined Randomizer. This is a solo Project made by Tkiwi18

Features I am aiming for:
1. Check Tracker
  1.1 Skip function
  1.2 Collected function
  1.3 Note function
2. Item Tracker
3. Entrance Tracker
4. Downloadable JSON file that saves your progress

Note. This Tracker WILL NOT connect with your rom in anyway meaning it IS NOT live updated.
You will have to control which items you have collected, checks you've checked, entrances you've checked
